# Restaurant Backend API

## Quick Start

1. Install: `npm install`
2. Run: `npm run dev`
3. API runs on: `http://localhost:5000`

## Demo Endpoints

- `GET /api/demo` - Project info
- `GET /api/health` - Health check
- `GET /api/demo-data/dashboard` - Mock stats
- `POST /api/auth/demo-login` - Demo login

## For Portfolio

This backend demonstrates:

- RESTful API design
- MongoDB integration
- JWT authentication
- CRUD operations
- Error handling

Note: All data is mock/sample for demonstration.
