# Restaurant Management System

A full-stack restaurant website with React frontend and Node.js backend.

## Features
- Menu display
- Table reservations
- Contact form

## Tech Stack
- React, HTML, CSS (Frontend)
- Node.js, Express (Backend)

## Setup
1. Clone repo
2. Backend: `cd backend && npm install && npm start`
3. Frontend: `cd frontend && npm install && npm start`
